Sample behavioral waveforms for design file count_dac.tdf

The following waveforms show the behavior of lpm_counter megafunction for the chosen set of parameters in design count_dac.tdf. The design count_dac.tdf is a 13 bit up counter. The counter has an asynchronous clear input.

Fig. 1 : Wave showing counter operation.

The output port cout will be asserted at the completion of count sequence. The ports cin and cout are used to chain multiple counters together.
